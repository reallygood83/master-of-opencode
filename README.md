### "Imagine Jarvis for your Notes"
Think of your Obsidian Vault as your **digital brain**. Until now, it was a quiet library where you stored thoughts. 
**Master of OpenCode** is like hiring a **genius active assistant** who lives inside that library. This assistant doesn't just read books (notes); it can write new ones, reorganize shelves, run calculations, and even build tools for you.

It bridges the gap between static notes and dynamic action using the power of the **OpenCode CLI**.

### âœ¨ Key Features
1.  **Universal Intelligence**: Why settle for one implementation? Switch instantly between **Claude (Anthropic), GPT-4 (OpenAI), Gemini (Google), or DeepSeek/Llama (Ollama)**. It's like having a panel of experts on speed dial.
2.  **Conversation Persistence**: Just like a real colleague, it remembers your previous conversations. You can pause a discussion today and resume it tomorrow exactly where you left off.
3.  **Context Aware**: Click "Send Active File," and the agent instantly understands what you're working on. No more copy-pasting code snippets.
4.  **Real Action (Agentic)**: This isn't just a chatbot. It connects to the OpenCode CLI to **execute terminal commands**, create files, and run builds directly from your chat interface.
